# Source tree notes

> Inherits from ../AGENTS.md

## Quick map

- `gateway/` - Telegram bot and HTTP/WS server
- `bridge/` - JSONL bridge, sessions, and command parsing
- `daemon/` - CLI entrypoint and daemon lifecycle
- `cron/` - scheduled jobs
- `protocol/` - shared types and protocol version
- `memory/` - qmd-based recall and memory tool
- `workspace/` - links and concepts indexes
- `client/` - Node client utilities
- `logging/` - file logging helpers
